# バナーサイズチェッカー

バナーのファイル名にサイズを入れて、ウィンドウにドラッグ＆ドロップしてください。
画像の縦横サイズをチェックして、正しくないと思われる箇所は赤文字で表示します。

## 使いかた

ファイル名は「banner_300x250.jpg」のように、幅→高さの順に数字を入れてください。
数字は3桁以上にしてください。100より小さい場合は頭に「0」を入れてください。
例）「90」→「090」

ファイル名からサイズが抜き出せない場合も、赤文字で表示します。

バナーファイルは、1つでもたくさんでも大丈夫。

画像としてロードできないファイルは無視。表示もしません。

## 仕様

ファイル名の中で、3桁以上の数字をサイズ指定とみなします。

数字の1つ目を横幅、2つ目を高さとして扱います。
